Google Earth for Linux

Linux (Install Googe Earth on Linux)

Google Earth should be installed using gdebi package install gdebi:

sudo apt install gdebi-core

download the Google Earth for linux package:

wget https://dl.google.com/dl/earth/client/current/google-earth-pro-stable_current_amd64.deb

install google earth using gdebi command

sudo gdebi google-earth-pro-stable_current_amd64.deb

Google Earth for Android

Android Apkpure Installer should be downloaded. Connect Android with USB cable, click "File Transfer", Copy downloaded files "google earth" and "apkpure installer" Android "Download" directory. From android File browser Install apkpure, Run apkpure, click "Me", and navigate to "google earth" .apkm or .xapk file and install.

Android Google Earth For 32-bit CPU search for google earth armeabi v7a apk download Download probably comes from Apkmirror where file name looks like "com.google.earth_10.91.0.3...apkmirror.apkm" For 64-bit CPU search for google earth arm v8a apk download Download probably comes from Apkmirror where file name looks like "com.google.earth_10.91.0.3...apkmirror.apkm" or Apkpure where file name looks like "Google Earth_10.91.0.3_APKPure.xapk"

Firefox click on .kmz

Configure firefox to open .kmz file with Google Earth just by a click on .kmz file

After installing Google Earth install .desktop files

cp google-earth-pro.desktop /home/$USER/.local/share/applications/

Check that it works, on gnome move mouse top left, on search type "google earth" and it should find Google Earth, click it and it should open.

When this works proceed to configure firefox:
